# A股宏观策略周报：全球风险偏好共振，内资驱动力凸显

**日期：** 2026-03-08
**分析师：** 宏观策略组

### 1. 市场情绪诊断：从“外热内冷”转向“内外共振”

*   **A股本土情绪：高亢且具韧性**
    上证指数已站稳4100点上方（4124.19），深证成指逼近14200点。成交量持续放大，反映出场内投资者已从“谨慎观望”转为“积极进攻”。这种情绪不仅源于指数层面的突破，更来自于对国内增长逻辑的确认。
*   **全球宏观情绪：极度Risk-On**
    美元指数（DXY）回落至98.85，有效缓解了新兴市场的流动性压力。纳指（22387.68）与标普500（6740.02）均处于高位，反映出全球资金对于科技成长和经济软着陆的极高风险偏好。
*   **情绪背离分析：**
    当前**本土与全球情绪呈现“罕见的同步性”**。以往常见的“美股创新高、A股独憔悴”的背离已经消失。这种共振意味着：A股已摆脱单一的防御逻辑，开始与全球科技牛市接轨。如果全球情绪回落，A股因内资占比提升和政策底支撑，表现出更强的“抗跌性”。

### 2. 核心市场表现与归因

*   **A股（上证4124 / 沪深300 4660）：内生流动性是核心驱动**
    A股的强势主要归因于**极度宽松的国内货币环境**。1.78%的中国10年期国债收益率与4100点的指数形成鲜明对比，资产荒压力迫使权益类资产估值中枢上移。
*   **美股映射：科技溢价的传导**
    纳斯达克的强势表现通过算力、半导体及AI产业链直接映射至A股相关板块。只要美股AI逻辑未破，A股的科技成长风格将持续获得全球估值锚的支撑。

### 3. 关键要素深度分析

*   **人民币汇率（6.896）：被动的强势与北向回归**
    尽管中美利差倒挂拉大，但美元/人民币汇率维持在6.90以下。这种“汇率稳、利率低”的组合极度有利于A股：既防止了资本外流，又为国内货币政策留出了空间。6.89的汇率水平正在吸引长线外资（配置型资金）回流沪深300核心资产。
*   **大宗商品（铜 5.8 / 原油 90.9）：再通胀预期抬头**
    铜价处于5.80的高位，原油冲上90美元，反映了全球工业需求复苏及供应侧扰动。这直接利好A股的**有色、能源及化工板块**。高铜价暗示全球制造业周期正在重启，对A股的中游制造板块构成业绩支撑。
*   **中美利差（-2.35%）：极度倒挂下的“资产荒”逻辑**
    美国10Y债率4.13% vs 中国1.78%。如此巨大的利差在历史上通常对应资金流出，但当前A股不跌反涨，说明**国内流动性溢价已覆盖了利差风险**。投资者更看重国内低利率环境下的分红率价值与成长股的PEG溢价。

### 4. 核心结论与策略展望

*   **一句话总结**：**机会远大于风险。** 当前处于“低利率+弱美元+强资源”的黄金窗口期。
*   **短期展望（1-3个月）**：
    上证指数有望在4000点上方筑牢新底部，并挑战更高空间。市场已从“估值修复”转向“景气驱动”。只要中国10年期国债收益率保持在2.0%以下，A股的流动性盛宴就不会结束。
*   **关注方向**：
    1.  **全球定价的资源品**：铜、铝及油气相关的资源板块，受益于再通胀逻辑。
    2.  **AI与硬科技**：受纳指高位运行的映射，关注国产半导体及智能驾驶。
    3.  **高股息资产**：在1.78%的无风险利率背景下，具备4%以上分红率的红利资产仍是底仓首选。

**风险提示：** 警惕地缘政治突发扰动导致原油暴涨引发的恶性通胀，以及美联储超预期鹰派风险。
