# A股市场策略周报：全球风险偏好回暖，内资驱动力凸显

## 1. 市场情绪诊断：共振中的强韧表现

*   **A股本土情绪：亢奋与韧性并存**
    上证指数站稳4000点大关（4083.97），深证成指逼近15700点。指数表现反映出场内投资者极高的风险偏好，成交量能维持高位，市场已脱离底部磨底阶段，进入显著的趋势性行情。
*   **全球宏观情绪：典型的“Risk-On”模式**
    *   **VIX指数 (15.39)**：处于历史低位区间，预示全球市场波动率收敛，恐慌情绪平息。
    *   **美元指数 (99.38)**：跌破100整数关口，美元走弱显著释放了新兴市场的流动性压力。
    全球资金目前处于积极扩张头寸的“冒险模式”。
*   **情绪背离分析（关键点）：**
    当前**本土情绪与全球情绪呈现“正向共振”而非背离**。值得注意的是，尽管中美利差倒挂严重，但A股表现出极强的“以我为主”特征。全球乐观情绪主要由科技增长预期驱动，而A股的强势则更多包含了本土政策溢价与经济复苏预期的自我实现。这种共振意味着短期内A股不存在外部流动性突变的冲击风险。

## 2. 核心市场表现与归因

*   **A股核心指数**：
    上证指数报收4083.97，CSI 300报收4938.81。
    *   **归因**：核心驱动力已从单纯的“政策博弈”转向“基本面复苏+资金面接力”。在低利率环境下，国内非银资金（险资、理财子）加速向权益类资产迁移，形成了显著的增量资金流。
*   **美股映射与传导**：
    纳斯达克（26912.78）与标普500（7593.22）维持强势。美股科技龙头的估值扩张对A股TMT、半导体及AI产业链形成直接的“估值天花板”上移效应，带动了深证成指的同步走强。

## 3. 关键要素分析

*   **人民币汇率 (USD/CNY 6.77)**：
    尽管中美10年期国债利差仍维持在约-276bps的深度倒挂状态（4.47% vs 1.71%），但人民币汇率逆势走强至6.77。这反映了**经常项目顺差的支撑及外资对人民币资产长期价值的重估**。强汇率环境抵消了利差倒挂对北向资金的抑制，有利于吸引长线国际资本配置A股。
*   **大宗商品与通胀预期**：
    *   **原油 (92.33)** 与 **铜 (6.545)** 维持高位，预示全球总需求并未陷入衰退，但也暗示了成本端压力依然存在。
    *   对A股影响：资源品、上游化工板块仍有业绩支撑，但在PPI向CPI传导的过程中，需警惕中下游制造业利润空间的挤压。
*   **中美利差 (倒挂程度：-276bps)**：
    国内10年期国债收益率低至1.71%，处于超低利率环境。这种极低的无风险收益率强制倒逼资金流向高股息（红利资产）及高成长（科创板）板块，是当前A股走出独立行情的核心底层逻辑。

## 4. 核心结论与策略展望

*   **一句话总结**：
    当前环境对A股投资者而言是**“机会显著大于风险”**，市场处于资金面充盈与全球风险偏好抬升的甜蜜期。
*   **短期展望 (1-3个月)**：
    上证指数在4000点上方虽有震荡需求，但趋势向上明确。随着半年报预期的开启，市场将从“情绪驱动”转向“业绩证伪”。预计指数将呈现震荡上行态势，4200点为下一阻力位。
*   **关注方向**：
    1.  **全球映射链条**：紧跟美股AI及半导体趋势，关注国内算力、自主可控及先进制程。
    2.  **“低利率”避风港**：在1.7%的国债收益率背景下，具备分红能力的红利低波资产仍是底仓首选。
    3.  **资源类龙头**：受益于原油、铜等大宗商品高位运行的有色、能源板块，具有抗通胀属性。
